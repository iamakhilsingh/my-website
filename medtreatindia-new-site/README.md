# MedTreat India New Site

Static replacement website for MedTreat India.

Open `index.html` in a browser to preview locally, or upload this folder to a static host and point the domain to it.

Important files:

- `index.html` - page structure and content.
- `about.html`, `treatments.html`, `hospitals.html`, `process.html`, `security.html`, `contact.html` - individual section pages.
- `testimonials.html`, `video-testimonials.html`, `written-testimonials.html` - patient story pages with separate video and written testimonial paths.
- `styles.css` - brand styling and responsive layout.
- `script.js` - country selector, mobile navigation and WhatsApp message builder.
- `assets/brand/` - optimized logo assets.
- `assets/images/` - local professional imagery.
- `assets/reference/` - the supplied website screenshots used as visual references for the cleaner layout direction.
- `_headers` and `security-headers.md` - hosting security header recommendations.

Launch checklist:

- Confirm the official social profile URLs.
- Confirm the company WhatsApp number inside `script.js`.
- Replace any placeholder-safe testimonial copy with approved real patient stories only after consent.
- Replace text-style hospital logo chips with official logo artwork if the hospitals approve/logo files are supplied.
- Enable HTTPS and the security headers in `security-headers.md`.
